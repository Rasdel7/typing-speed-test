# Typing Speed Test ⌨️

Test your typing speed and accuracy with
WPM tracking, difficulty levels and leaderboard.

## Live Demo
[Click here](https://typing-speed-test-55y6ucggejdaezkpnhisoy.streamlit.app)

## Features
- 3 difficulty levels: Easy, Medium, Hard
- Words and Sentences mode
- Live WPM and accuracy tracking
- WPM gauge with skill levels
- Persistent leaderboard
- Personal progress charts

## Tools Used
- Python, Streamlit, Pandas, Matplotlib

## How to Run Locally
pip install streamlit pandas matplotlib
streamlit run app.py
